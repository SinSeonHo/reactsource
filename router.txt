라우팅 : 사용자가 요청한 URL에 따라 알맞은 페이지 보여주기
SPA(Single Page Application) 이지만 여러 개의 URL로 다른 뷰를 보여줘야 할 때 사용

1) React router
    - 리액트 라우팅관련 라이브러리 중 가장 많이 사용됨
    - 컴포넌트 기반으로 라우팅 시스템 설정 (경로설정)

    main.jsx 최상위에서 BrowserRouter로 감싸야됨
    a태그대신 Link를 사용

    




2) Next.js
    - 리액트 프로젝트의 프레임워크
    - 리액트 프로젝트 설정, 라우팅 시스템, 최적화, 다국어 시스템 지원, 서버 사이드 렌더링 등 지원
